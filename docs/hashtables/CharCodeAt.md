# Using CharCodeAt & Modulus Operator


## `charCodeAt()`
Returns an integer representing the UTF-16 code unit at the given index.
```javascript
var str = 'hello world';
var result = str.charCodeAt(1);
console.log(result); = // 101
```
## Modulus Operator
Returns the remainder when dividing two values.
```javascript
console.log(7 % 3); // 1
```
